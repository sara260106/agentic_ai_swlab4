# Assignment 3: LLM-Based Agent

## Objective
To integrate a language model for intelligent decision-making.

## Description
This agent uses an LLM (simulated) to decide which tool to use.

## Features
- LLM-based decision making
- Tool execution
- Logging system

## Tools
- Calculator
- Weather
- Summarizer

## Architecture
Input → LLM Decision → Tool Execution → Output → Logging

## Learning Outcomes
- Prompt-based reasoning
- AI-driven tool selection
- Logging and traceability

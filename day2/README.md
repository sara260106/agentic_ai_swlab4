# Assignment 2: Tool-Using Agent

## Objective
To enable the AI agent to use external tools (functions).

## Description
This agent extends a rule-based system by introducing tool usage.

## Tools Implemented
1. Calculator Tool – performs arithmetic operations
2. Weather Tool – provides mock weather data
3. Summarizer Tool – summarizes text

## Architecture
Input → Intent Detection → Tool Selection → Tool Execution → Output

## File Structure
- tools.py → Contains all tools
- agent.py → Controls agent logic

## Learning Outcomes
- Tool abstraction
- Function calling
- Modular programming
